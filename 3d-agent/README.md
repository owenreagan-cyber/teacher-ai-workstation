# 3D Design Agent Readiness

The 3D Design Agent is a future first-class specialist system. It is separate from, but coordinated by, the Teacher AI Chief of Staff.

It will support classroom, hobby, personal, repair/replacement, reference-only, and business 3D printing. It may eventually help with product ideation, design briefs, OpenSCAD/CadQuery generation, printability checks, slicer preparation, product listings, prototype logs, failure diagnosis, and business product memory.

Phase 0C is groundwork only. The agent does not currently control printers or slicers, send print jobs, or approve commercial use automatically.

The future system can warn, classify, recommend review, and help with checklists. It cannot block, stop, disable, prevent, or authorize printing. The human operator always makes the final print decision.

## Current tools

- Bambu Studio: default slicer for Bambu printers.
- OpenSCAD: initial AI-friendly, text-based, parametric CAD path.

## Current boundaries

- No printer-control automation.
- No Bambu cloud/network automation.
- No marketplace scraping.
- No commercial use assumptions for downloaded/reference models.
- No "print-ready" claims for AI-generated designs.
- Use "slicer-ready pending human verification" only after checklist review.
