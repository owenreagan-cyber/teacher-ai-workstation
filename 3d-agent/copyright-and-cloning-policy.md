# Copyright and Cloning Policy

Feature toggles are advisory preferences, not enforcement controls.

```json
  "default_use_mode_question_required": true,

  "software_blocks_prints": false
}
```

## Principles

- Personal-use inspiration/reference can be allowed.
- Personal/reference use does not automatically mean commercial use is safe.
- Reference-Only is a filing and warning category, not a hard restriction.
- Commercial products should be reviewed by a human before sale.
- The agent can recommend original alternatives when a commercial idea resembles protected IP.
- The agent can warn about commercial clone risk or trademarked character sale risk.
- The agent cannot block, stop, disable, or prevent printing.
- The human operator makes the final print decision.
