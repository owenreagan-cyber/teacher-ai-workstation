# Design Principles

- Ask clarifying questions before generating a design.
- Always ask whether the use mode is personal, classroom, business/commercial, repair/replacement, or reference-only.
- Prefer parametric designs.
- Prefer simple printable geometry first.
- Create v1 prototypes before complex final models.
- Track dimensions, constraints, filament, printer, and print time.
- Every design should include a printability check.
- Every child/classroom item should include a safety check.
- Every commercial item should include IP/reference notes.
- Designs should be versioned.
- The agent advises and documents; it does not block or authorize printing.

## Versioning starter

- `v0.1`: first generated sketch or placeholder.
- `v0.2`: first dimensional correction.
- `v1.0`: first human-reviewed design ready for slicer inspection.

Use "slicer-ready pending human verification" only after dimensions, geometry, safety notes, and reference notes are reviewed.
