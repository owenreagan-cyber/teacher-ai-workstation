# 3D Agent Training

Training the 3D agent means improving prompts, examples, verification habits, and product memory from real outcomes.

## Training sources

- Successful design examples
- Failed prints and why they failed
- Contrastive pairs of good/bad CAD outputs
- Repeatable OpenSCAD test prompts
- Prototype result logs
- Local and cloud model comparisons
- Actual Bambu Studio and print outcomes

## Rule

No model output goes directly from prompt to print without verification.
