# 3D Verification

Every AI-generated 3D design should pass human-confirmed verification before being used in Bambu Studio.

Verification is advisory and human-confirmed, not automatic. The agent must never call a design "print-ready." It may only call a design "slicer-ready pending human verification."

Verification should include geometry, dimensions, color/multicolor setup, supports, safety, IP/reference notes, and Bambu Studio preview checks.

The system can warn, classify, and recommend review, but it cannot block, stop, disable, or prevent a print. The human operator is responsible for the final decision to print.
