# Color and Multicolor

- For AMS/multicolor prints, color regions should be separate bodies/shells where possible.
- Color maps must identify body/part and intended filament slot/color.
- Clarify whether color is separate mesh bodies, Bambu Studio paint-on color, filament swap by layer, or SVG/layered extrusion.
- Warn about purge waste and color transitions.
- Require human verification in Bambu Studio layer/color preview.
- Flag if color regions are ambiguous or share faces in a way that may cause bleeding/artifacts.

The color/multicolor checklist is advisory and does not block printing.
