# Dimension Accuracy

- Always ask for target dimensions in mm before generating a functional design.
- Confirm width x depth x height.
- Flag inferred dimensions.
- Include expected dimension summary after generation.
- Require verification in Bambu Studio before printing.
- For fit-based items, require clearance/tolerance notes.

## Examples

- Pencil hole: ask for pencil diameter before assuming fit.
- Nameplate: ask for max desk/plate size.
- Fidget: ask for age range and target hand size.

The checklist is advisory and does not block printing.
