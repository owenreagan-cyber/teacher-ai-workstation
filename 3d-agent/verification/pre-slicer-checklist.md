# Pre-Slicer Checklist

Use this before importing or printing any AI-generated design.

- [ ] Use mode selected: personal / classroom / business / repair-replacement / reference-only
- [ ] IP/reference notes reviewed if commercial or reference-based
- [ ] Safety notes reviewed if child/classroom item
- [ ] Dimensions specified in mm
- [ ] Expected dimensions verified
- [ ] Mesh/geometry appears valid
- [ ] Bottom/orientation checked
- [ ] Overhangs/support strategy reviewed
- [ ] Color/multicolor plan reviewed if relevant
- [ ] File saved to correct folder
- [ ] Bambu Studio printer/profile selected
- [ ] Bambu Studio scale checked
- [ ] Bambu Studio layer preview inspected
- [ ] Bambu Studio support preview inspected
- [ ] Bambu Studio color/AMS preview inspected if relevant
- [ ] Human operator reviewed warnings before printing
- [ ] Human operator made final print decision
- [ ] Human review completed before sale if business product

The checklist is advisory. It does not block printing.
