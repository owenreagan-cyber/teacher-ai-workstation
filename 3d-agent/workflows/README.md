# 3D Agent Workflows

These workflows are planning tools for a future 3D Design Agent. They are beginner-friendly and advisory.

Every workflow should include:

- Purpose
- Inputs needed
- Steps
- Output format
- Safety/IP/printability footer where relevant

## Standard footer

- Use mode: personal / classroom / business-commercial / repair-replacement / reference-only
- Printability notes
- Safety notes
- IP/reference notes
- Human review recommended
- Human operator final decision
- Next action
