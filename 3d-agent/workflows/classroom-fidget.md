# Classroom Fidget Workflow

## Purpose

Plan classroom fidgets, prizes, manipulatives, nameplates, and teaching tools with safety notes.

## Inputs needed

- Student age range
- Classroom purpose
- Target size in mm
- Material
- Durability needs
- Quantity
- Use mode confirmation

## Steps

1. Confirm classroom use.
2. Ask age range and handling expectations.
3. Check small parts, sharp edges, pinch points, fragile joints, and material suitability.
4. Prefer simple support-light geometry.
5. Prepare a prototype-first design brief.

## Output format

- Classroom item concept
- Safety notes
- Dimensions
- Prototype plan
- Bambu Studio checks

## Footer

- Use mode: personal / classroom / business-commercial / repair-replacement / reference-only
- Printability notes
- Safety notes
- IP/reference notes
- Human review recommended
- Human operator final decision
- Next action
