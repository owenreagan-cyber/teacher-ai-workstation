# IP Safety Check Workflow

## Purpose

Classify a 3D idea by use mode and provide advisory IP/reference warnings.

## First question

Is this design for:

- Personal use
- Classroom use
- Business/commercial use
- Repair/replacement
- Reference-only

## Reference questions

Is the design based on:

- An existing product
- Character
- Logo
- Brand
- Franchise
- Downloaded model
- Marketplace listing
- Customer-provided file
- Photo/image reference

## Steps

1. Ask the first question and reference questions.
2. Classify the use mode.
3. Summarize source/reference notes.
4. Add commercial-use warning if needed.
5. Add reference-only warning if needed.
6. Suggest an original alternative if relevant.
7. Remind Owen that the human operator makes the final decision.

## Output format

- Use mode classification
- Source/reference notes
- Commercial-use warning if needed
- Reference-only warning if needed
- Suggested original alternative if relevant
- Human review recommendation
- Final decision reminder

## Footer

- Use mode: personal / classroom / business-commercial / repair-replacement / reference-only
- Printability notes
- Safety notes
- IP/reference notes
- Human review recommended
- Human operator final decision
- Next action
