# Print Failure Diagnosis Workflow

## Purpose

Diagnose common print failures and recommend next checks.

## Inputs needed

- Exact failure description
- Photos if available
- Printer/profile
- Material
- Nozzle size
- Bambu Studio settings
- Model version

## Common issues

- Stringing
- Warping
- Weak joints
- Supports
- Poor tolerances
- Bad fit
- Surface defects

## Steps

1. Ask for photos and exact settings when available.
2. Classify the likely failure type.
3. Separate model geometry issues from slicer/material/printer issues.
4. Recommend a small next test.
5. Log the result for future product memory.

## Output format

- Likely issue
- Evidence
- Next diagnostic check
- Suggested v2 change or slicer check
- What to record

## Footer

- Use mode: personal / classroom / business-commercial / repair-replacement / reference-only
- Printability notes
- Safety notes
- IP/reference notes
- Human review recommended
- Human operator final decision
- Next action
