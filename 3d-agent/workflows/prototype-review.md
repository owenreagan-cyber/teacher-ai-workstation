# Prototype Review Workflow

## Purpose

Review a printed prototype and propose v2.

## Inputs needed

- Product/design name
- Version
- Photos if available
- Print settings
- Material
- What worked
- What failed
- Dimensional measurements

## Steps

1. Record the current version.
2. Compare target dimensions to measured dimensions.
3. Identify print defects or fit problems.
4. Propose one focused v2 revision.
5. Update the prototype log.

## Output format

- Prototype summary
- Measurement notes
- Failure notes
- v2 changes
- Next test

## Footer

- Use mode: personal / classroom / business-commercial / repair-replacement / reference-only
- Printability notes
- Safety notes
- IP/reference notes
- Human review recommended
- Human operator final decision
- Next action
