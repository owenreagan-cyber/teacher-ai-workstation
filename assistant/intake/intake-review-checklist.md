# Intake Review Checklist

Before promoting any intake item, confirm:

[ ] I know what this material is.
[ ] I know where it came from.
[ ] I know why it should be used.
[ ] It does not include student private data.
[ ] It does not include parent private data.
[ ] It does not include passwords, API keys, tokens, or credentials.
[ ] It does not include confidential school records.
[ ] It does not include unnecessary full copyrighted text.
[ ] It has a clear target use.
[ ] It has been summarized or sanitized if needed.
[ ] It has been logged in intake-log.md.
[ ] Owen has made the final approval decision.

If any answer is uncertain, choose "needs more context" or "quarantine," not approval.
