# Intake Quarantine

## Purpose

Track materials that should not be used until manually reviewed.

Quarantine is for material that appears sensitive, unclear, private, copyrighted, risky, or not ready. Quarantined material must not be used as assistant context.

Quarantine means review needed, not deletion. Owen decides what happens next.

## Quarantine Reasons

- possible student/private data
- possible parent/private data
- possible credential or secret
- possible confidential school record
- possible copyrighted material
- unclear source or license
- unclear permission
- unclear use case
- too raw or messy to use safely
