# Lesson Brief Drafts

This folder stores local lesson brief drafts.

Drafts are for teacher review only and may be created by `scripts/create-lesson-brief.sh`.

## Safety rules

- Do not include student-sensitive data.
- Do not include real student names.
- Do not include grades, medical notes, behavior notes, IEP/504 details, or parent contact information.
- Human review is required before classroom use.
- No briefs are sent, shared, uploaded, synced, or published by these scripts.
- Connected services require explicit future permission.

Draft `.md` files in this folder are gitignored, except this README.
