# Lesson Planning Queue

## Purpose

This queue tracks generic lesson planning ideas that may become future local lesson briefs, activities, assessments, or materials checklists.

This is not a lesson generator. It is a safe planning queue for local teacher use only.

## Safety Warning

Do not include real student names, private student data, grades, medical information, behavior notes, IEP/504 details, or parent contact information.

Use generic topics, source notes, and next actions. Lesson drafts require human review before classroom use.

## Allowed Status Values

- idea
- drafted
- reviewed
- ready
- archived

## Lesson Brief Helper Note

Future helpers may use this queue, but the current Lesson Brief Helper uses a safe slug only and does not parse queue rows yet. Keep queue entries generic and safe for future review.
