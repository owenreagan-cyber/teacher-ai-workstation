# Personal Review Notes

This folder is for Owen's optional personal human review notes after inspecting a lesson slug.

## Safety rules

- Local teacher notes only.
- Notes are optional.
- Notes are not official review status.
- Do not include student-sensitive data.
- Do not include real student names.
- Do not include grades, medical notes, behavior notes, IEP/504 details, or parent contact information.
- Human review is required before classroom use.
- No notes are sent, shared, uploaded, synced, or published by these scripts.
- Connected services require explicit future permission.

## How to use

1. Copy `docs/review-notes-template.md` into this folder manually if you want a local note file.
2. Name the file with a safe lesson slug, for example `fractions-review-notes.md`.
3. Keep personal note files local. Do not commit them unless Owen explicitly chooses to.

See `docs/review-notes-template.md`.
