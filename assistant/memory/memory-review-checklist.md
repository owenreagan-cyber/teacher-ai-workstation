# Memory Review Checklist

Before including memory in a Chief of Staff run, confirm:

- [ ] Memory files are intentionally included.
- [ ] Ran `bin/chief-of-staff --memory-status`.
- [ ] Ran `bin/chief-of-staff --validate-memory`.
- [ ] Checked for stale memory markers in `--show-context`.
- [ ] Confirmed `memory-log.md` does not need an update.
- [ ] Confirmed writing-style confidence is still accurate.
- [ ] No student private data is included.
- [ ] No parent private data is included.
- [ ] No passwords, API keys, or credentials are included.
- [ ] No confidential school records are included.
- [ ] Stale facts are labeled "needs review."
- [ ] User's latest instruction overrides old memory.
- [ ] Safety, source, and permission rules override memory.
- [ ] Memory was included with an explicit CLI flag.
