# Preference Memory

## Purpose

Track stable preferences that help the Chief of Staff reduce repeated questions.

## Tool Preferences

- Local-first where practical.
- Human-in-the-loop before automation.
- Markdown-readable docs before databases.
- Bambu Studio as default Bambu slicer.
- OpenSCAD as initial AI-friendly CAD path.
- CLI before app UI.

## Workflow Preferences

- Keep roadmaps, checklists, phases, and completion status visible.
- Give specific beginner-friendly steps.
- Avoid hiding uncertainty.
- Do not move too quickly into automation.
