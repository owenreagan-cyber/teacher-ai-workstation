# Contrastive Pairs

## Purpose

Teach the assistant by showing a weak version beside a better version.

## How to use

Add anonymized examples that explain what makes one response preferred. Do not include student names, grades, IEP details, discipline details, or private contact info.

## Starter template

```markdown
## Pair label

### Avoid

Bad or less useful example.

### Prefer

Better example.

### Why

- Tone:
- Structure:
- Accuracy:
- Safety:
```
