# Feedback Log

## Purpose

Capture corrections and preferences so the assistant improves over time.

## How to use

Add short entries whenever Owen edits, rejects, or corrects assistant output. Do not paste private student-sensitive content here.

## Starter template

```markdown
## YYYY-MM-DD - Short label

- Task:
- What the assistant did:
- What Owen changed:
- Pattern to remember:
- Should this update writing-style.md, a workflow, or memory policy?
- Sensitivity:
```
