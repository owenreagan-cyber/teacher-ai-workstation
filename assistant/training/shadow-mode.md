# Shadow Mode

## Purpose

Compare assistant output against Owen's final version before trusting automation.

## How to use

Run the assistant in draft-only mode. Owen completes the real version manually. Compare the two and score whether the assistant is becoming more useful.

## Starter template

```markdown
## YYYY-MM-DD - Shadow comparison

- Task:
- Assistant draft:
- Owen final:
- Score 1-5:
- What matched:
- What missed:
- Safety concerns:
- Update needed:
```
