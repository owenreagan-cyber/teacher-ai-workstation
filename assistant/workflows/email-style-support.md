# Email Style Support Workflow

## Purpose

Draft emails in Owen's approved style.

## Inputs needed

- Audience
- Purpose
- Key facts
- Tone
- Approved writing samples, if style matching is requested

## Steps

1. Confirm this is draft-only.
2. Ask for approved writing samples if style matching is needed.
3. Draft a clear email.
4. Flag sensitive details and missing facts.

## Output format

- Subject
- Draft email
- Optional shorter version
- Notes for Owen

## Safety notes

- Never send email.
- Use approved writing samples only.
- Do not include student-sensitive details unless explicitly approved and necessary.

## Required footer

- Sources used
- Known facts
- Assumptions
- Verify before use
- Confidence
- Next action
