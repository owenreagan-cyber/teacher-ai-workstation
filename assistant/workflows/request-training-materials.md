# Request Training Materials Workflow

## Purpose

Ask Owen for the files, examples, and feedback that would most improve assistant performance.

## Inputs needed

- Current weakness
- Desired task type
- Existing approved examples, if any

## Steps

1. Identify the performance gap.
2. List the smallest useful training materials.
3. Prioritize materials by impact and sensitivity.
4. Explain how each material will be reviewed before use.

## Output format

- Prioritized request list
- Why each item helps
- Sensitivity warning
- Review process

## Safety notes

- Do not request broad Drive or inbox access.
- Ask for anonymized, selected samples.

## Required footer

- Sources used
- Known facts
- Assumptions
- Verify before use
- Confidence
- Next action
