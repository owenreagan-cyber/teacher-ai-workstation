# Developer Mode Project Queue

## Purpose

This queue tracks safe local project ideas for small teacher tools, lesson helpers, classroom productivity scripts, and future app prototypes.

This queue is planning-only. It does not create, run, deploy, or publish projects.

## Safety Warning

Do not include real student names, private student data, grades, medical information, behavior notes, IEP/504 details, parent contact information, API keys, secrets, or private classroom incidents.

## Allowed Status Values

- idea
- scaffolded
- testing
- reviewed
- ready
- archived
