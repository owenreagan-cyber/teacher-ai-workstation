# 3D Printing Day 1 Setup

Bambu Studio is the default slicer for Bambu printers.

OpenSCAD is installed for AI-friendly code-based CAD. It is text-based, parametric, and easy for an assistant to generate and review.

OrcaSlicer can be explored manually later if desired, but it is not required for Day 1.

## Folder layout

- `~/3D-Printing/Business`
- `~/3D-Printing/Classroom`
- `~/3D-Printing/Personal`
- `~/3D-Printing/Intake`
- `~/3D-Printing/Reference-Only`
- `~/3D-Printing/Licenses`
- `~/3D-Printing/Prototype-Logs`
- `~/3D-Printing/Product-Catalog`

Keep commercial products separate from reference/personal items.

Do not assume downloaded models are licensed for sale. Reference-Only exists for inspiration, personal projects, and non-commercial references. Business work should use original designs or properly licensed sources.

## Tool roles

- OpenSCAD: AI-friendly design code.
- Bambu Studio: scale, supports, colors, printer profile, layer preview, and print estimates must be verified here.

No AI-generated file should be treated as print-ready on Day 1. Use `3d-agent/verification/pre-slicer-checklist.md` before printing AI-generated designs.

The checklist is advisory and does not block printing.
