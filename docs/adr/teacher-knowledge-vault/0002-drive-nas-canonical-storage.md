# ADR-0002: Google Drive / UGREEN NAS Canonical Storage Model

**Status:** Accepted (M0 freeze)  
**Date:** 2026-07-04  
**Context:** Multiple devices and Canvas create storage confusion.  
**Decision:** Google Drive primary canonical; UGREEN NAS mirror/backup; MacBook Pro M5 workshop/staging only; Canvas publishing only.  
**Consequences:** Reconciliation compares against Drive/NAS authority.  
**Blocked runtime:** Mac-as-canonical auto-sync; Canvas-as-storage.  
**Validation hooks:** `docs/teacher-knowledge-vault/canonical-storage-and-taxonomy.md`
