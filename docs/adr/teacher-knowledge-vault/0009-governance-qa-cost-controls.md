# ADR-0009: Governance, QA Gates, and Cost Controls

**Status:** Accepted (M0 freeze)  
**Date:** 2026-07-04  
**Context:** Multi-engine system needs build/architecture/product gates.  
**Decision:** Three-level QA gates; observability includes cost and blocked-operation counts.  
**Consequences:** Future CI aligns to documented gates.  
**Blocked runtime:** Weakening gates without ADR.
