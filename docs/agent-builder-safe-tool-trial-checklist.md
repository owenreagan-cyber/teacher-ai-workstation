# Agent Builder Safe Tool Trial Checklist

Last updated: 2026-07-03

```text
Status: planning_checklist_only
Authority: not implementation approval
Use before: any experimental external agentic tool trial (Claude Code, Antigravity, future tools)
```

## Pre-Trial

- [ ] Tool classified as **experimental** in `docs/agent-builder-compatibility-and-external-tool-governance.md`
- [ ] Owen explicitly authorizes a bounded trial mission
- [ ] Trial scope is docs/status only OR repo-local with full PR lifecycle — no runtime product scope
- [ ] Student data prohibition acknowledged
- [ ] Real curriculum ingestion blocked
- [ ] Production registry parked state verified (`records` count 1; sentinel intact)

## During Trial

- [ ] No API keys, OAuth, or network integrations added to repo
- [ ] No Chief of Staff commands invoke the external tool
- [ ] No Mac system changes or tool installs committed to repo
- [ ] External planning input filed as classification only (not repo authority)
- [ ] PASS does not authorize implementation banner present in status output

## Post-Trial Proof

- [ ] Mission proof report filed per `docs/agent-builder-mission-proof-template.md`
- [ ] Dashboard and validate-all: 0 FAIL
- [ ] Proposal ledger updated if tool promoted to planning-only or approved builder
- [ ] Blocked runtime doc updated if new risks discovered

## Escalation

Stop and report if trial would require API/OAuth, student data, real curriculum, registry writes, or runtime generation.

## Non-Activation

Checklist only. Does not approve tool use.
