# Appearance Guide

Teacher OS uses a two-mode visual design. Both modes use the same macOS user account.

## Teacher Workstation Mode

Teacher Workstation Mode should feel:

- clean
- minimal
- academic
- low distraction
- productivity-first Dock
- professional wallpaper

Use this mode for lesson planning, Canvas work, curriculum notes, and printable output review.

## Casual Anime Mode

Casual Anime Mode can include:

- anime
- music
- streaming
- relaxed visuals
- personal browser profile
- anime wallpaper

Use Focus mode, wallpaper, widgets, browser profiles, and manual Dock preferences to make this mode feel different while still using the same Mac account.
