# Browser Profiles Guide

Use separate browser profiles so school work and personal browsing stay visually and mentally separate.

## Work browser profile

Create a Work profile in Chrome or Arc. Add bookmarks for:

- Canvas
- Thales
- Google Drive
- ChatGPT
- Gemini
- GitHub
- Canva
- Supabase
- Lovable
- Cursor docs

## Casual browser profile

Create a Casual profile in Chrome or Arc. Add bookmarks for:

- YouTube
- streaming
- music
- anime
- personal AI use

## Tip

Use different profile colors and names so it is obvious which mode you are in.
