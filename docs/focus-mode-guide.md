# Focus Mode Guide

Focus setup is manual because macOS does not expose a reliable public command-line interface for creating full Focus profiles.

## Teacher Workstation Mode

Create a Focus mode named **Teacher Workstation Mode**.

Recommended setup:

- Use a professional wallpaper.
- Add Calendar, Reminders, Weather, and Battery widgets.
- Allow school and productivity notifications.
- Hide entertainment distractions.

## Casual Anime Mode

Create a Focus mode named **Casual Anime Mode**.

Recommended setup:

- Use an anime wallpaper.
- Add Music, Photos, and Weather widgets.
- Allow personal and media notifications.
- Hide school notifications.

## Suggested steps

1. Open **System Settings**.
2. Open **Focus**.
3. Create each Focus mode.
4. Configure allowed people, apps, and schedules.
5. Pair each Focus mode with its wallpaper and widgets.
6. Enable iPhone/iPad Focus sync if desired.
