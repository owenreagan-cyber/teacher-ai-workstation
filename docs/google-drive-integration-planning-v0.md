# Google Drive Integration Planning v0

Last updated: 2026-07-01

```text
Status: planning only
Drive API: inactive
OAuth: inactive
Sync jobs: inactive
```

## Purpose

Plan a future **manual-reference-first** Google Drive integration seam for curriculum library metadata. Teacher Workstation stores references to Drive locations; it does not crawl, scan, or sync Drive by default.

## Future Scope (Approval-Gated)

- manual Drive link/ID metadata entry after explicit approval
- teacher-reviewed reference verification workflow
- optional future read-only metadata refresh mission (separate approval)

## Current Boundaries

- no Google Drive API activation
- no OAuth or credential storage
- no folder scanning or automatic indexing
- no network calls

See `docs/integration-planning-foundation-v0.md` and `docs/curriculum-source-storage-strategy.md`.
