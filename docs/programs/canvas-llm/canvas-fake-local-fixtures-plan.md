# Canvas Fake Local Fixtures Plan

```text
Status: PHASE_0_DOCS_ONLY
Classification: fake/local fixtures only
Real Canvas content: blocked
```

## Fixture Purpose

Fake/local fixtures may illustrate evidence categories and report shapes without touching Canvas or real curriculum.

Allowed fixture examples:

- fake page metadata.
- fake module metadata.
- fake assignment metadata.
- fake HTML patterns with invented text.
- fake self-healing candidate categories.

Blocked fixture content:

- copied Canvas pages, modules, assignments, discussions, submissions, comments, rosters, grades, or analytics.
- real student data.
- real course content.
- real curriculum excerpts.
- exported Canvas files.

## Fixture Location

Use `fixtures/canvas-llm/` for fake/local examples. Every file must clearly say fake/local and non-runtime.
