# Canonical Medical Center Diagnostics

## Status

Canonical planning specification.

## Purpose

The Medical Center answers:

```text
What could go wrong before Canvas is touched?
```

## Diagnostic Levels

```text
PASS
WARN
FAIL
BLOCKED
```

## Required Diagnostic Panels

Future Medical Center should include:

- Page Standard Check
- Assignment Gatekeeper
- Resource Integrity
- Reading/Spelling Integrity
- Math Test Integrity
- Newsletter Boundary
- Teacher Memory Confidence
- Canvas Brain Confidence
- Course ID Drift
- File Operation Risk
- Write Gate Readiness

## Diagnostic Rules

Diagnostics must run before write approval.

Diagnostics must explain:

- what was checked
- source evidence
- result level
- reason
- required human action if any

## Boundary

Medical Center diagnostics are read-only until a future write-gate phase explicitly permits operations.

Diagnostics do not authorize Canvas writes by themselves.
