# Canonical Newsletter Rules

## Status

APPROVED_PATTERN

## Canonical Model

```text
Newsletter = Homeroom Canvas Page
Announcement = notification only
```

## Canonical Announcement Text

```text
The newsletter has been updated for the week of {date range}.
```

## Newsletter Page Owns

- weekly updates
- family communication
- events
- reminders
- approved birthday handling

## Announcement Role

The announcement only notifies families that the newsletter page has been updated.

It does not replace the newsletter page.

## Birthday Privacy Rule

Birthday information requires separate privacy approval before reuse or storage.

Birthday names must not become automatic teacher memory or persistent generated content without approval.

## Implementation Boundary

This rule does not authorize:

- newsletter page creation
- announcement creation
- publishing
- student data use
- birthday storage
