# Canonical Reading / Spelling Together Rules

## Status

APPROVED_PATTERN

## Rule

Reading and Spelling belong to a unified literacy workflow.

## Canonical Identity

```text
Reading prefix = RM4
Spelling prefix = RM4
```

Spelling uses RM4 because it shares Reading course identity.

## Canonical Course/Page Model

Reading and Spelling:

- may have separate pacing rows
- may have separate assignments
- share RM4 course identity
- render on the Reading-owned Canvas page
- use combined announcements
- do not create standalone Spelling pages

## Spelling Announcement Rule

Standalone Spelling announcements are blocked.

Spelling communication should be included in combined Reading/Spelling communication.

## Future Assignment Policy

Spelling lessons do not automatically create daily assignment buckets.

Spelling Tests may create assignments according to reviewed test rules.

## Required Future Tables

Before implementation, create reviewed tables for:

- Spelling Test coverage
- Spelling word bank
- focus word rule
- Reading Test mapping
- Reading Checkout rule
- Reading homework rule
- combined announcement template
