# Production Weekly Page Doctrine

Real Canvas weekly pages are preloaded at the beginning of the year.

Production automation updates existing QxWy pages, preserves the HTML shell, replaces only editable content regions, publishes or schedules the page for Friday at 4:00 PM for the following week, and sets the page as the course front page.

Production pages are updated, not created.

Sandbox experiments may create temporary test pages in course `24399`, but that does not change production doctrine.
