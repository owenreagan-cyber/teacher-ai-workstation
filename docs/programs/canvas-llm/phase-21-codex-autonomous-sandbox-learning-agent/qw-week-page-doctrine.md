# Q/W Week Page Doctrine

`Q` means quarter.

`W` means week within that quarter.

Weeks reset each quarter and track.

Correct sequence:

```text
Q1W8
Q1W9
Q2W1
```

Incorrect sequence:

```text
Q1W10
```

The agent must identify existing preloaded weekly pages, titles, and slugs before proposing any update. It must not infer a global week counter.
