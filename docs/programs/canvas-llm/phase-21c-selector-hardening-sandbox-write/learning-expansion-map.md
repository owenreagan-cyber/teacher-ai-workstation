# Phase 21C Learning Expansion Map

## Announcements

Learn announcement patterns without sending notifications.

Future rules should map:

- weekly update announcement
- snow day/test date change announcement
- newsletter update announcement
- parent-safe summary language

## Attachments and Files

Learn attachment/file patterns without mutating files.

Future rules should map:

- weekly page file links
- assignment attachment links
- Power Up resources
- Math practice resources
- Reading/comprehension resources

## Assignments

Learn assignment naming and relationship patterns.

Future rules should map:

- Math lesson assignment
- Math Power Up assignment/practice
- Math fact test
- Math study guide
- Reading lesson assignment
- Reading comprehension letter
- Reading page number
- spelling/reading shared RM4 behavior

## Safety

Learning artifacts remain local and ignored.

No grade, student, people, submission, or gradebook data may be read.
