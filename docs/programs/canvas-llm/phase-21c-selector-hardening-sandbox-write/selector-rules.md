# Phase 21C Existing-Page Selector Rules

## Required Target

The selector must only target:

```text
course_id: 24399
```

## Required Page Type

The target must be an existing page.

The selector must not create a page.

## Preference Order

The selector should prefer:

1. normal QxWy page
2. unpublished page
3. non-front-page
4. non-special page
5. non-copy page

## Avoid Unless Explicitly Requested

The selector should avoid:

- Q1END
- Q3END
- Q4W10
- current front page
- copied/duplicate pages
- reference course pages
- production course pages

## Write Meaning

A write is only allowed after preview and approval.

A write means updating one existing sandbox page body only.
