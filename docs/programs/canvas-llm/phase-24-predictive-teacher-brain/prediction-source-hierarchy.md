# Prediction Source Hierarchy

Authority order:

1. owner-confirmed hard rules
2. explicit current-year pacing-guide entry
3. approved teacher correction
4. repeated FPK pacing-guide pattern
5. predictive suggestion
6. unresolved

Hard rules always win. Historical patterns may inform suggestions, but they never override owner-confirmed or explicit current-year guidance.

Every prediction should report which layer supplied the decision and whether review is required.
