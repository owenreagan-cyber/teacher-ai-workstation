# Resource Safety Classification

Phase 25 treats resource identity, verification, availability, visibility, deployment policy, and Canvas readiness as separate dimensions.

Visibility classes:

- student-facing
- teacher-only
- answer-key
- assessment-secure
- local-preview-only

Deployment policy classes:

- approved-for-canvas-link
- approved-for-future-upload
- link-only
- local-only
- blocked
- unknown

Canvas status is metadata only and never triggers a write.
