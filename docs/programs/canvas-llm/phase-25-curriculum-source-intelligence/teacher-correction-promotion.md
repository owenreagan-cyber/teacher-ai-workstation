# Teacher Correction Promotion

Phase 25 extends the local correction-memory pattern from Phase 24.

Supported correction scopes:

- this occurrence only
- this week
- this lesson
- lesson range
- parity family
- assessment family
- curriculum sequence
- global reusable resource

Occurrence-only corrections stay occurrence-only unless a teacher explicitly promotes them. Contaminated state is quarantined instead of being silently rewritten.
