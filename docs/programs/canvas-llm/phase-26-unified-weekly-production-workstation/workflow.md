# Workflow

The teacher workflow is:

1. Select a week.
2. Load the current pacing state.
3. Generate predicted instruction.
4. Resolve curriculum resources.
5. Review unresolved or conflicting items.
6. Apply teacher corrections.
7. Regenerate the week.
8. Preview pages, assignments, reminders, and resources.
9. Approve subject by subject.
10. Approve the full week.
11. Export the local production package.

The UI keeps the teacher inside one app for the common weekly loop.
