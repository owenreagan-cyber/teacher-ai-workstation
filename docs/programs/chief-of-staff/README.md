# Chief Of Staff Program

```text
Status: active orchestration/readiness layer
Classification: documentation/status and read-only workflow guidance
Runtime/product writes: blocked unless approved
```

Chief of Staff is the status, orchestration, safety, readiness, and proof layer for Teacher AI Workstation. It may report status, surface blockers, guide next actions, and preserve PASS/WARN/FAIL proof semantics.

It must not authorize blocked implementation, bypass approval gates, unfreeze Canvas LLM runtime work, perform production writes, handle student data, or activate live integrations.
