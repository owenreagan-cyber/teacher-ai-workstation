# Local-First Curriculum Library Plan

```text
Status: active planning
Classification: local-first metadata/reference architecture
```

The Curriculum Library should be built around local metadata records and source references.

Preferred stored fields:

- stable local IDs.
- source location references.
- file hashes or metadata hashes where approved.
- tags and curriculum relationships.
- review status.
- safety status.
- source links.
- notes about blocked/approved handling.

Blocked unless separately approved:

- real curriculum ingestion.
- OCR or text extraction.
- embeddings or RAG.
- general folder scanning.
- production/canonical registry writes.
- student data.
