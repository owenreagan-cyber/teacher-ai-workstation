# Source Reference Registry Plan

```text
Status: planned
Classification: metadata/reference-only
```

The source reference registry should describe where curriculum assets live without making the repo the owner of large files.

Allowed planning fields:

- source type.
- source label.
- path or link reference when approved.
- access status.
- review status.
- last metadata check date.
- safety status.
- relationship to lessons or resource packages.

Blocked by default:

- copying source files.
- ingesting source content.
- scanning unapproved folders.
- connector/API/OAuth access.
- production writes.
