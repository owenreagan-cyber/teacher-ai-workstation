# Teacher Knowledge Vault Metadata Schema

Minimum resource record:

```json
{
  "id": "resource-sm5-study-guide-12-blank",
  "canonicalName": "SM5_Study_Guide_012_Blank_v1.pdf",
  "originalName": "original filename.pdf",
  "subject": "math",
  "curriculum": "saxon-math-5",
  "resourceType": "study-guide",
  "variant": "blank",
  "lessonNumbers": [],
  "testNumbers": [12],
  "powerUpCodes": [],
  "audience": "student",
  "sensitivity": "student-facing",
  "schoolYears": ["2026-2027"],
  "sha256": "sha256-value",
  "mimeType": "application/pdf",
  "storageLocations": [],
  "canvasReferences": [],
  "verificationStatus": "verified",
  "canonicalStatus": "active",
  "supersedes": null
}
```

Required operational metadata:

- stable resource ID
- original name
- canonical name
- SHA-256
- MIME type
- file size
- canonical storage location
- mirror and reference locations
- subject
- curriculum
- lesson, test, and unit relationships
- resource type
- resource variant
- audience
- sensitivity
- verification status
- duplicate group
- source provenance
- school year
- curriculum edition
- created timestamp
- updated timestamp
- last verified timestamp
- move and rename revision history
