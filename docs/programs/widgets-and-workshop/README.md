# Widgets And Workshop Program

```text
Status: planning/status track
Classification: no install, no Mac changes
```

Widgets, shortcuts, wallpapers, visual workshop, and 3D/workshop ideas remain planning/status unless separately approved.

Blocked unless separately approved:

- widget install.
- shortcut install or execution.
- Mac system changes.
- wallpaper runtime changes.
- CAD/slicing/printing.
- automation/scheduler.
- external integrations.
