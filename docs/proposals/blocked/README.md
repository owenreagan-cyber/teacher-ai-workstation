# Blocked (Approval-Gated)

Last updated: 2026-07-02

## Purpose

Planning docs for work that is **explicitly blocked** until Owen Reagan or a scoped implementation prompt approves.

Typical blockers: production writes, APIs/OAuth, student data, real curriculum intake, Mac system changes, local model inference, live integrations.

## Rules

- Classify risk and required approval level in each doc
- Do not imply implementation approval
- Link to `docs/implementation-approval-gate.md` and domain boundaries
