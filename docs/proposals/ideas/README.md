# Ideas (Unclassified / Early)

Last updated: 2026-07-02

## Purpose

Scratch space for early ideas **before** they earn a row in `docs/proposals/index.md`.

## Rules

- Planning-only — no implementation scaffolds
- No student data, real curriculum content, or secrets
- Promote to ledger when value/risk/approval level are clear
- Check `docs/proposals/index.md` for duplicates before promoting

## Lifecycle

```text
ideas/ → ledger row (proposed) → approved_for_* → implementation mission → implemented/
```
