# Raycast Setup

Raycast is installed automatically by the Brewfile, but Raycast preferences may need manual setup after the first launch.

## First launch

1. Open Raycast from Applications or the Dock.
2. Follow the onboarding prompts.
3. Set the preferred hotkey.
4. Enable useful extensions only when you need them.

## Suggested commands

Create or configure commands for:

- Open Teacher OS repo
- Open Canvas
- Open Obsidian Vault
- New Agenda Project later
- New Worksheet Project later
- Start Teacher Mode guide
- Start Casual Mode guide
