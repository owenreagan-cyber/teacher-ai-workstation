# Reference-Only Content Policy

Reference-Only means a file may be useful for inspiration, personal wallpaper, private study, or local planning, but it is not approved for public, classroom, or product use.

## Reference-Only by default

- Anime character art.
- Fan art.
- Reddit images.
- Pinterest-style inspiration.
- Screenshots.
- Images with unknown source or license.

## Allowed local uses

- Personal Mac wallpapers.
- Personal Photos widgets.
- Private mood boards.
- Local creative inspiration.
- Visual reference while brainstorming.

## Uses that require review first

- Public branding.
- Product images or product designs.
- Listing images.
- Public classroom slide decks.
- School websites or handouts.
- Public social posts.

## Review path

A Reference-Only item can move to another use only after human review.

Possible review states:

```text
approved-for-personal-wallpaper
approved-for-teacher-use
approved-for-presentation-use
approved-for-product-use
rejected
```

Until reviewed, the default remains:

```text
Reference-Only. Human review required before broader use.
```

## Intake connection

Downloaded images are candidate material. They are not approved material.

Future tooling may generate intake stubs from the source manifest so Owen can review whether any item should stay Reference-Only or move to an approved use.
