# Teacher Knowledge Vault — Document Understanding Pipeline

Last updated: 2026-07-04

## Escalation Ladder

0 metadata → 1 native extraction → 2 local OCR → 3 rules → 4 local AI (future) → 5 cloud AI (approval-gated).

## Cost Rules

Metadata before extraction; native before OCR; local OCR before cloud; page OCR before full doc; cache by hash; no bulk background OCR; no cloud without cost gate.
