# Manual Exported Evidence Inbox

```text
Status: manual_local_inbox
Authority: non-authoritative until reviewed
Automatic scanning: blocked
Student data: blocked
Credentials/secrets: blocked
```

This folder is for future manually placed, approved, redacted Canvas evidence. Nothing in this folder is automatically trusted or promoted.

Before placing anything here, remove:

- student names, IDs, submissions, grades, comments, messages, accommodations, rosters, analytics, or family/private details.
- Canvas API tokens, OAuth files, client secrets, cookies, session data, or environment files.
- full unreviewed course exports.
- real curriculum folders.

This folder must remain manual/local only. Do not add scripts that automatically scan, crawl, fetch, import, or summarize it.
