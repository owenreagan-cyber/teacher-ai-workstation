# Canvas LLM Fake Local Fixtures

```text
Status: fake_local_only
Classification: non-runtime examples
Canvas API/OAuth/live reads: blocked
Student data: blocked
```

This folder is for invented examples that help Phase 0 describe Canvas evidence categories. These fixtures are not Canvas exports and must never include real course content or student data.

Allowed:

- fake page/module/assignment metadata.
- fake HTML patterns with invented classroom-neutral text.
- fake self-healing candidate categories.

Blocked:

- live Canvas data.
- copied Canvas content.
- student names, grades, submissions, comments, rosters, or analytics.
- real curriculum excerpts.
- generated deployment payloads.
